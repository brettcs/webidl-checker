Attribute InheritsGetterOnNonReadOnlyAttribute.name inherits a getter but is not readonly.
Attribute InheritsGetterOnInterfaceWithoutInheritance.name inherits a getter but does not belong to an interface with inheritance.
Attribute InheritsGetterWithIncompatibleType.name inherits a getter but does not have the same type (double) as the attribute Base.name (DOMString) it inherits from.
